# Userspace RCU

This crate provides unsafe Rust API to [`liburcu-bp`][liburcu] for Linux systems.

This crate should not be used directly, please use the [`urcu2`] crate.

[liburcu]: https://liburcu.org/
[`urcu2`]: https://crates.io/crates/urcu2
